# 國中寫作 AI 備課指令系統

整合**台灣108課綱**第四學習階段學習表現與**國中教育會考評分規準**的國中語文教師 AI 備課工具。

## 適用對象

- 國中七至九年級語文教師（第四學習階段）
- 適合搭配 ChatGPT 或 Claude 使用

## 核心特色

**課綱 × 會考雙軸整合**——每個模組同時對照108課綱學習表現（6-Ⅳ）與會考評分規準四個向度（立意取材、結構組織、遣詞造句、錯別字格式標點），幫助教師在素養導向與會考實務之間找到最佳教學路徑。

## 功能模組

| 模組 | 名稱 | 說明 |
|------|------|------|
| A | 學情診斷與課程定位 | 對照課綱指標與會考級分，診斷學生能力層次與審題能力 |
| B | 單元寫作教學設計 | 依年段、文體、節數與會考題型方向生成完整教學計畫 |
| C | 審題鷹架與立意深化 | 針對六種引導式題型設計審題工具、立意深化提問、分層任務 |
| D | 會考導向評量與回饋 | 對照六級分規準生成評量規準、AI批閱指令、自評互評工具 |

## 內建六種會考引導式題型

1. 圖表＋文字說明型
2. 自訂題目填空型
3. 閱讀引文＋立場選擇型
4. 圖片＋分段寫作任務型
5. 情境應用文書信型
6. 圖文情境開放型
7. 圖像＋詞彙選擇詮釋型（114年新樣態）
8. 多圖文卡片情境型（115年新樣態）

## 使用方式

1. 開啟網頁，選擇對應模組
2. 填寫基本教學資訊
3. 點擊「生成指令」按鈕
4. 複製生成的指令，貼入 ChatGPT 或 Claude 使用
